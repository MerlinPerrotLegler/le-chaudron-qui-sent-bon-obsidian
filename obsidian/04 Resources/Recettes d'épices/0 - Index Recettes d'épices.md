# Index — Recettes d'épices (mélanges)

Recettes des **mélanges d'épices** du Chaudron qui Sent Bon. Chaque fiche précise : **nom**, **équipements** nécessaires à la confection (hors conditionnement), **ingrédients** avec liens vers la [[0 - Index Plantes Aromatiques|fiche plante]] (ou vers [[Produits d'importation]] si non produit), **proportions** et **procédé**.

> Démarche : pour chaque mélange, une version **authentique** (fidèle au classique, import assumé) et, quand c'est pertinent, une version **terroir** maximisant le « cultivé à la ferme » (substituts locaux — voir [[1 - Épices non cultivables en Aveyron]]).
> Réglementation par type de produit : voir [[0 - Cadre général]] et fiches [[Mélanges d'épices secs]] · [[Sels aromatisés]] · [[Sucres aromatisés]] · [[Préparations en saumure et lactofermentées]].

## Équipements du labo (récapitulatif)

| Équipement | Usage |
|-----------|-------|
| **Séchoir / déshydrateur** | Sécher plantes, graines, zestes, piments avant transformation |
| **Fumoir** | Fumage (paprika fumé, sels fumés) |
| **Batteuse** | Battre/égrener les tiges sèches (herbes, ombellifères) pour séparer feuilles/graines |
| **Broyeur / moulin** | Réduire en poudre graines, baies, piments (réglage grossier→fin) |
| **Tamis** | Calibrer la mouture, retirer tiges et grosses particules |
| **Torréfacteur / poêle** | Torréfaction légère des graines (cumin, coriandre, sésame) avant broyage |
| **Marmite / cuve chauffante** | Infusion et dissolution du sucre (sirops) |
| **Étamine / filtre + tamis fin** | Filtration des infusions (sirops) |
| **Réfractomètre (Brix)** | Contrôler la teneur en sucre des sirops (≥ 55 %) |
| **Pasteurisateur / stérilisateur** | Stabiliser sirops et conserves |
| **Cuve / bocaux de fermentation + poids** | Lactofermentation, saumures (cornichons, choucroute) |
| **pH-mètre / bandelettes** | Vérifier l'acidification des saumures (pH < 4,5) |
| **Balance de précision** | Peser les proportions |

*Phase de conditionnement (pesée finale, mise en pot/sachet, étiquetage) non comptée ici — voir [[Labo PPAM]].*
