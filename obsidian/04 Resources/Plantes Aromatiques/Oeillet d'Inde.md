# Œillet d'Inde (*Tagetes patula*)

Annuelle à fleurs orange-jaune. Puissante plante compagne répulsive (nématodes, aleurodes). Utilisée dans les mélanges aromatiques du Chaudron. Fleurs comestibles.

## Données culturales

| Paramètre | Valeur |
|-----------|--------|
| Famille | Astéracées |
| Cycle | Annuel |
| Hauteur | 20-50 cm |
| pH optimal | 6,0 – 7,5 |
| Sol | Léger, drainant, modérément fertile |

## Exposition

**Plein soleil.** Chaleur et lumière nécessaires. Très résistant à la chaleur estivale.

## Besoin en eau

Faible à modéré. Arrosage modéré.

## Culture

- **Semis** : sous abri mars-avril, repiquage en mai.
- **Espacement** : 20-30 cm.
- **À planter en bordure** de toutes les cultures.

## Risques de culture

| Risque | Description | Prévention |
|--------|-------------|------------|
| Limaces | Jeunes plants | Cendres, pièges |

## Associations favorables

- **Toutes les cultures** : répulsif universel contre nématodes et aleurodes
- [[Basilic]], tomates : association classique de jardin
- Entre rangées de PPAM : répulsif général

## Transformation en épice / usage

### Récolte
Cueillir les fleurs épanouies régulièrement.

### Séchage
- Sur claies, 35°C.
- **Ratio de séchage : 0,20** (perte 95 % eau).

### Usage dans les recettes du Chaudron
- Fleurs séchées : mélanges de tisanes, colorants naturels, décoration de plats
