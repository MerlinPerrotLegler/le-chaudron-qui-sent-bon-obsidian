## Associations à retenir

### Associations favorables majeures

- **Trio garrigue** : [[Thym]] + [[Romarin]] + [[Lavande]] + [[Sarriette]] + [[Hélichryse]]
- **Jardin médicinal** : [[Camomille Romaine]] + [[Mauve]] + [[Guimauve]] + [[Mélilot]]
- **Compagnes protectrices** : [[Souci]] + [[Oeillet d'Inde]] + [[Grande Ortie]] (en bordure)
- **Cuisine fraîche** : [[Basilic]] + Tomates + [[Persil]] + [[Estragon]]

### Incompatibilités majeures à retenir

> ⚠️ **Le fenouil est allélopathique** — à isoler de toutes les autres cultures aromatiques.
> ⚠️ **La menthe** est incompatible avec les plantes de garrigue (besoins en eau opposés).
> ⚠️ **L'ail** inhibe les légumineuses ([[Fenugrec]], [[Mélilot]]).
