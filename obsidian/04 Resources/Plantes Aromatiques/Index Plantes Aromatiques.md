# Index des Plantes Aromatiques & Épices — Le Chaudron qui sent bon

Référentiel complet des plantes cultivables à **La Cavalerie de Combret** (12370, Sud-Aveyron, Rougier de Camarès) pouvant produire des épices, aromates, tisanes ou condiments. Chaque fiche détaille : **culture, exposition, besoin en eau, risques, associations favorables/défavorables, transformation en épice**.

**89 fiches de plantes** + cette note de référence sur les [[Épices non cultivables en Aveyron]].

> Démarche : pour chaque épice utilisée (fichier *Recettes et production.xlsx*), on est remonté à la **plante d'origine** afin de déterminer si elle peut être produite sur la ferme.

## 🤝 Associations — synthèse

### Associations favorables majeures
- **Trio garrigue** : [[Thym]] + [[Romarin]] + [[Lavande]] + [[Sarriette]] + [[Hélichryse]] + [[Hysope]] + [[Sauge sclarée]]
- **Safranière** : [[Safran]] avec aromatiques de garrigue (sol sec drainant commun)
- **Jardin frais & médicinal** : [[Mélisse]] + [[Menthe]] + [[Guimauve]] + [[Reine des prés]] + [[Angélique]]
- **Sous-bois aromatique** : [[Cerfeuil musqué]] + [[Aspérule odorante]] + [[Grande Ortie]]
- **Compagnes protectrices** (bordures) : [[Souci]] + [[Oeillet d'Inde]] + [[Capucine]] + [[Tanaisie]] + [[Achillée millefeuille]]
- **Faux-poivres maison** : [[Poivre de Sichuan]] + [[Poivre de Timut]] + [[Gattilier]] + [[Genévrier commun]]

### ⚠️ Incompatibilités majeures à retenir
- **[[Fenouil]]** : allélopathique — à **isoler** de presque toutes les cultures.
- **[[Absinthe]], Artemisia** : allélopathiques (absinthine) — isoler.
- **[[Menthe]], [[Grande Ortie]], [[Houblon]], [[Raifort]], [[Réglisse]], [[Tanaisie]]** : **traçantes/envahissantes** — confiner (bac, barrière anti-rhizome).
- **[[Ail]], [[Oignon]], [[Échalote]]** : inhibent les **légumineuses** ([[Fenugrec]], [[Mélilot]], [[Réglisse]]).
- **Plantes de garrigue sèche vs plantes de sol frais** : ne jamais mélanger (besoins en eau opposés).
- **Confusions toxiques dangereuses** : [[Gentiane jaune]] (vs vératre), [[Berce commune]] (vs berce du Caucase), [[Angélique]] (vs ciguë), [[Laurier sauce]] (vs laurier-rose/cerise), [[Sumac]] (vs sumac vénéneux), [[Safran]] (vs colchique). **Identification rigoureuse impérative.**
