# Plan Opérationnel — Pilotage quotidien

Outil de pilotage concret pour l'exploitant. Moins de théorie, plus d'action : **quoi faire, quand, comment mesurer**.

| Mois | Priorité labo | Priorité culture | Priorité commercial | Jalon financier |
|------|--------------|-----------------|--------------------|-----------------|

| **Dec** | Ventes Noël max. Bilan recettes. | Repos. Commandes plants an 2. | Foires Noël. E-mailing clients. | Bilan comptable an 1. |

## Checklist démarrage (An 0 → An 1)

### Administratif
- [ ] Immatriculation MSA (Centre de Formalités des Entreprises)
- [ ] Choix statut : EARL / SCEA / Entreprise individuelle agricole
- [ ] Ouverture compte bancaire pro
- [ ] Souscription assurance responsabilité civile professionnelle
- [ ] Démarche certification BIO (organisme : Ecocert / Bureau Veritas / CERTIPAQ)
- [ ] Dossier DJA (si éligible < 40 ans)
- [ ] Dossier PCAE Occitanie (équipements)
- [ ] Adhésion Chambre Agriculture Aveyron
- [ ] Adhésion Erboressens / S.I.M.P.L.E.S. (pour débouchés PPAM)

### Labo
- [ ] Déclaration d'activité labo de transformation (DDPP / DDCSPP)
- [ ] Mise aux normes hygiène labo (HACCP)
- [ ] Référencement DLC/DLUO pour chaque produit
- [ ] Création étiquettes conformes (mentions obligatoires : ingrédients, allergènes, poids net, DLC, numéro lot, exploitant)
- [ ] Test recettes × 3 avant commercialisation
- [ ] Stock emballages initial (6 semaines de tampon)

### Production
- [ ] Analyse de sol parcelle PPAM (pH, MO, oligo-éléments)
- [ ] Commande plants / semences an 1 (lavande, thym, hélichryse)
- [ ] Planification irrigation (retenue collinaire ou branchement)
- [ ] Matériel minimal : déshydrateur, balance, claies de séchage
- [ ] Partenaire distillation à façon (lavande, si alambic non disponible an 1)

### Commercial
- [ ] Création identité visuelle (logo, palette couleur)
- [ ] Shooting photos produits
- [ ] 3 marchés identifiés et réservés
- [ ] 5 épiceries/boutiques bio démarchées
- [ ] Page Instagram / Facebook créée

### Gestion des lots

À chaque production, noter :
- Date de fabrication
- Numéro de lot (ex: HDP-2026-07-001)
- Quantité produite (unités + poids brut matières)
- DLC (durée de conservation)
- Coût matières premières du lot
